# The restaurant list
Discover your new favourite restaurants: [The restaurant list](https://restaurant-list-lime.vercel.app/)

## Instructions

### Server / Rest API:
Create a .env file and include the following environment var:
```bash
PORT
MONGO_URI
DOMAIN_REMOTE
DOMAIN_LOCAL
TOKEN_SECRET
CLOUDINARY_NAME
CLOUDINARY_KEY
CLOUDINARY_SECRET
```
